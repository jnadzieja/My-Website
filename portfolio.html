New York Times Search

This is an app I made with my group that can search the New York Times API for your search terms and return up to 10 articles. This app also includes a date range feature to allow searching through specific ranges. Handy for finding older archived articles.

Train Time

This is an app I made for tracking the arrival times of trains. You can add your own information and it will do all the math for you. It could also be used for other methods of transportation, it's not necessarily limited to trains.

TravelGroupie

This was my first project as part of the bootcamp, I developed this on a team with 3 others. We made a site that allowed users to search for concerts through TicketMaster API, and while searching for concerts also get flights to concert destinations and high rated restaurants in the area of the concert.

Web Store

A web store I've begun working on in my free time, it's something I wanted to code just for fun. Eventually I would love to do something with it, and get a back-end working and everything.

Craft-A-Crawl

The second project I completed while in the UNC Charlotte coding bootcamp. On this project I primarily worked on front-end. I was responsible for the layout of the website as well as the button functionality and front-end server requests. We developed this app to be mobile first, but it will work well on any platform. This app can be used to create or join bar crawls, and we really wanted to make it as user friendly as possible. Our idea for this came from the fact that there really isnt a good way to go about doing bar crawls, and we wanted our web app to have a purpose.

Employee Data Management

An app that can be used to do basic data management for employees using the data you enter. It will calculate the months an employee has been working, and how much they are paid each month to see their total payout.

GifTastic

This is a small app I made to display a series of gifs based on a button click. The button you click searches for the term in giphy's API and returns the most popular gifs with that tag right now.

LIRI Bot

LIRI Bot is a Node.js program I wrote to interact with APIs to provide an easy way to search for upcoming concerts, songs, and movies.

Bamazon

Bamazon is a Node.js app I wrote that takes user inputs to interact with a mock database used to buy products. It will update the database each time something is purchased, and show the total purchase cost.

FriendFinder

The first Node.js server app I made. It is an app that lets the user take a survey and matches them with someone with similar answers. It is the first full-stack app I have made and I am still working out the (small) bugs and math used to match people.

Eat-Da-Burger

This is a Node.js and handlebars app that uses bootstrap for styling and uses a MySQL database for storing data. With this app you can type in the name of a burger, hit submit and Node.js will communicate with the MySQL database to store the new information then use redirect to send you to the page with updated information on it.

Trivia Game

This is a simple trivia game I made, it only has four questions and is themed around my favorite movies. Not very complex, but one of the more fun apps I've made.
